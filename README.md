# Skill Sync

## About

Skill Sync is a web-based project that I developed to solve a simple but real problem — people have skills but often don’t know where to use them or how to find the right opportunities.

This system allows users to enter their skills and get relevant suggestions based on what they know. The main idea is to make skill utilization easier and more practical.

## How it works

First, the user registers or logs into the system.
After logging in, the user can enter their skills.

The backend processes the entered skills and matches them with available options or suggestions.
Right now, the matching is done using basic logic, but it still gives useful results.


## Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Python (Flask)
* Database: MongoDB

## Features

* User authentication (Login & Register)
* Skill input and storage
* Skill-based matching system
* Simple and easy-to-use interface

## How to Run

1. Clone the repository
2. Install all required libraries (Flask, pymongo, etc.)
3. Connect MongoDB properly
4. Run the Flask app
5. Open in browser (localhost)
